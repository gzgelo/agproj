# Angelo Gozo - Personal Portfolio Website

## Project Overview
This is my personal portfolio website as a second-year BSIT student at Camarines Norte State College. It showcases my skills, projects, and contact information.

**Live Website:** https://gzgelo.github.io/agproj/portfolio-website/

---

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Google Fonts

---

## APIs Integrated (3)

| API | Purpose |
|-----|---------|
| **GitHub API** | Displays my public repositories |
| **EmailJS API** | Sends emails from contact forms |
| **Leaflet/OpenStreetMap** | Shows my location on a map |

---

## Transaction Features (2)

1. **Contact Form** - Visitors can send me messages
2. **Project Inquiry Form** - Visitors can inquire about specific projects

Both forms include:
- JavaScript validation
- Email notifications via EmailJS
- Success/error toast messages
