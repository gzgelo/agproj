# AquaGuard: Smart Underwater Waste Detection & Collection System Using IoT 

## Description
An innovative underwater device cleaning system prototype created for Human-Computer Interaction (HCI) class. Focuses on user-centered design and functionality.

## Features
- 3D modeled underwater cleaning device
- User-centered design approach
- Ergonomic considerations
- Safety features for underwater use
- Visual prototype with Blender

## Technologies Used
- **Blender** - 3D modeling and design
- **UI/UX Design Principles** - User interface design
- **HCI Principles** - Human-computer interaction concepts
- **Prototyping Tools** - Design and testing
- **User Testing** - Feedback and iteration

## Design Process
1. **User Research** - Understanding user needs
2. **Wireframing** - Initial design concepts
3. **3D Modeling** - Blender design and rendering
4. **Prototyping** - Creating interactive prototype
5. **User Testing** - Gathering feedback
6. **Iteration** - Improving based on feedback
