# Dental Clinic Management System

## Description
A comprehensive clinic management system for appointment scheduling, patient records, and billing using text file database storage.

## Features
- Patient registration and management
- Appointment scheduling
- Billing and payment processing
- Treatment records
- Text file database (no SQL required)
- Search and filter functionality

## Technologies Used
- **Java** - Core programming language
- **Text File Database (.txt)** - Data storage
- **Java Swing** - Desktop GUI framework
- **File I/O Operations** - Read/write data to text files
- **Java Collections** - Data management in memory

## File Structure
- `patients.txt` - Stores patient information
- `appointments.txt` - Stores appointment details
- `billing.txt` - Stores billing records
- `users.txt` - Stores user credentials

1. Install Java JDK 8 or higher
2. Compile all Java files
3. Run the main class
4. Text files will be automatically created on first run
